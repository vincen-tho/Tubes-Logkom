## Deskripsi Program
- Membuat sebuah _farm simulation role-playing game_ sebagai seorang programmer yang harus bertani untuk dapat keluar dari dunia _game RPG_
- Permainan dibuat dalam **bahasa Prolog (GNU Prolog)** dengan menggunakan materi yang telah dipelajari di mata kuliah (IF2121)

## Prasyarat
> **Pastikan branch repository berada di `main`** </br>

**Clone repository ini menggunakan command berikut (git bash)**
```
$ git clone https://github.com/vincen-tho/Tubes-Logkom.git
```

## Run Program
**Run program di GNU Prolog dengan command sebagai berikut**
```
    [main].
```
